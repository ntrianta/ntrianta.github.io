Traffic Analysis Visualization

My MSc Thesis. In this project I explored certain visualization types applied on Network Traffic Data. The visualizations were designed exploiting several pre-attentive attributes as well as Gestalt principles of perception in order to convey as much interesting information as possible in the least amount of time. This project was carried out in co-operation with Fox IT, thus the created graphs were optimised for Security Operations. The tools I used were d3.js for visualization,
